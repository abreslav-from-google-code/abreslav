Реализуйте маленький язык программирования анимации.
Из входного файла считываются команды, они выполняются подряд, пока файл не закончится. Когда файл закончился, программа ждет нажатия любой клавиши для завершения.

Проверьте работоспособность программы на своих примерах, не только на дном предложенном в задании. Примеры входят в материал для сдачи.

Входной файл: input.txt
Каждая строка файла представляет собой команду:
  create создает объект
  move перемещает объект
  sleep осуществляет задержку

Синтаксис команд

create <тип> <имя> <x> <y> <R>
  <тип> - тип фигуры (circle - круг, square - квадрат)
  <имя> - имя фигуры, не содержит пробелов
  <x> - координата x центра
  <y> - координата y центра
  <R> - радиус (для квадрата - половина стороны)

move <имя> <x> <y>
  <имя> - имя ранее созданной фигуры, не содержит пробелов
  <x> - новая координата x центра
  <y> - новая координата y центра

sleep <t>
  <t> - время задержки в миллисекундах
